# Icon Tile Badge

A presentational component to be used with `IconTile`, rendering a an icon
badge.

<!-- STORY -->

## Basic Import

```js
import {
  IconTile,
  IconBadge,
} from '@lifeomic/chroma-react/components/IconTile';
```

## Basic Usage

### Specifying an icon

```jsx
<IconTile>
  <IconTileBadge icon={Edit2} />
</IconTile>
```

### Class Name

A class name can be provided.

```jsx
<IconTileBadge className="mr-4" />
```

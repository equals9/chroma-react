# Table Module Testing

The Table Module exports data-test attributes to make testing easier.

<!-- STORY -->

## Import

```js
import { testIds } from '@lifeomic/chroma-react/components/TableModule';
```

## Test Ids

The following test Ids are exported:

- Header Row
- Header Cell
- Body Row
- Body Cell
- Is Loading
- No Results
